# Codebase Overview

This repository is currently a **profile repository** centered around a single `README.md`.

## What this repo is

- A personal GitHub profile README for `amide-init`.
- Not an application/library codebase (no `src/`, build files, package manifests, or test suite at this time).

## Current structure

- `README.md`: The main and only source file. Contains:
  - Professional summary and current roles
  - Tech stack and skills snapshot
  - Highlighted project links
  - Current focus areas
  - Contact links and profile stats widgets

## Important things to know

1. **Single-source content repo**
   - All meaningful content is in markdown prose in `README.md`.
2. **External project references**
   - Most technical depth lives in linked external repositories (e.g., Gitfolio, SQL2NoSQL, Luxe-Edit, mask-detection).
3. **No local runtime**
   - There is currently nothing to build or run locally in this repo.

## What to learn next

For a newcomer who wants to go deeper:

1. Start with linked projects in `README.md`:
   - `gitfolio` for GitHub API + CI/CD workflows
   - `sql2nosql` for data modeling/migration design
   - `luxe-edit` for editor architecture and frontend extensibility
2. Explore domain tracks mentioned in the README:
   - AI/LLM product integration patterns
   - Full-stack architecture (.NET Core + React + GCP)
   - MLOps/DevOps teaching and production workflows
3. If this repo evolves into a multi-file project, add:
   - `CONTRIBUTING.md`
   - Development setup docs
   - Folder-level architecture notes
